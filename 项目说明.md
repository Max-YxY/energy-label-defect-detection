# 能效标签缺陷检测系统

> 基于 YOLOv8n 的工业级缺陷检测系统，部署于 STM32N647 + 树莓派 4B 平台，用于生产线能效标签的实时缺陷检测。

---

## 系统架构

```
┌──────────────────────┐      UART (115200)      ┌───────────────────────┐
│    Raspberry Pi 4B   │ ◄──────────────────────►│    STM32N647 + LCD    │
│                      │   L<等级>,<缺陷>,<位置>  │                       │
│  ┌───────────────┐   │                         │  ┌─────────────────┐  │
│  │ CSI Camera    │   │                         │  │ 4.3" RGB LCD    │  │
│  │ (IMX219)      │   │                         │  │ (800×480)       │  │
│  └───────┬───────┘   │                         │  └─────────────────┘  │
│          │           │                         │                       │
│  ┌───────▼───────┐   │                         │                       │
│  │ YOLOv8n ONNX  │   │                         │                       │
│  │  320×320      │   │                         │                       │
│  │  ~7 FPS       │   │                         │                       │
│  └───────────────┘   │                         └───────────────────────┘
└──────────────────────┘
```

## 核心功能

### 1. 缺陷检测（YOLOv8n）
- 10 类目标检测：5 级能效等级 + 污渍/破损/褶皱 + 标签/箱子
- 模型：YOLOv8n (Nano)，输入 320×320
- 精度：**99.69%**（9 类评测）
- 推理速度：树莓派 4B 上约 **7 FPS**

### 2. 树莓派端
- 摄像头实时采集 + ONNX Runtime 推理
- 时间平滑（3 帧）减少误报
- UART 串口发送结果到 STM32
- 协议：`L3,OK,OK` / `L2,STAIN+DAMAGE,DEV`

### 3. STM32 端
- 接收串口数据，实时显示在 4.3" LCD 上
- OK → 绿色背景，NG → 红色背景
- 支持等级 + 缺陷类型 + 位置偏移显示

### 4. PC 调试工具
- `simulate_monitor.py` — 模拟检测程序，自动发送 OK/NG
- `serial_send.py` — 手动串口调试
- `pi_send.py` — 一键发送协议命令
- `stress_test.py` — 压力测试，测 STM32 响应

## 类别定义

| ID | 类别 | 说明 |
|----|------|------|
| 0 | level_1 ~ level_5 | 能效等级 1~5 级 |
| 5 | stain | 污渍缺陷 |
| 6 | damage | 破损缺陷 |
| 7 | wrinkle | 褶皱缺陷 |
| 8 | label | 标签区域 |
| 9 | box | 箱子区域 |

## 运行效果

### LCD 显示效果
- **OK** → 绿色背景，显示 "PASS 等级:3"
- **NG** → 红色背景，显示 "FAIL 污渍+破损"
- 无检测 → 黑色背景

### 检测示例
| 输入 | 检测结果 |
|------|----------|
| 正常标签 | level_3, OK |
| 有污渍标签 | level_2, STAIN |
| 破损标签 | level_1, DAMAGE |
| 褶皱标签 | level_3, WRINKLE |

### 评估图表
- **混淆矩阵**：9 类 99.69% 准确率
- **训练曲线**：250 epochs，mAP50 收敛至 0.995
- **容差分析**：位置偏差容忍度 0.18 (x) / 0.155 (y)
