# energy-label-defect-detection
Energy Label Defect Detection — Real-time defect detection for energy efficiency labels using YOLOv8n on STM32N647 + Raspberry Pi 4B. Detects 5 energy grades and 3 defect types (stain/damage/wrinkle) over UART at ~7 FPS. Achieves 99.69% accuracy on 320×320 ONNX models.
